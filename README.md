# To - Do Application

A clean, modern, and highly interactive task management dashboard built with React.js. This application moves away from traditional, cluttered interfaces to provide a streamlined, classic light-themed user experience for daily productivity.

---

## 🚀 Live Demo
> **Live Link:** `https://your-username.github.io/classic-todo-app/`

---

## ✨ Features

- **Classic Light Theme:** A soft, high-contrast, elegant aesthetic using clean slate typography and deep indigo highlights designed to minimize eye strain.
- **Dynamic Categorization:** Tag your tasks into specific categories such as **Work**, **Personal**, **Shopping**, or **Others** using refined, color-coded pastel pills.
- **Native Drag & Drop Board:** Seamlessly move tasks from **Pending** to **Completed** columns using the native HTML5 Drag and Drop API—no bloated third-party libraries required.
- **Tactile UI Experience:** Smooth lift-on-hover card transformations and striking visual indicators for finished items (reduced opacity and emerald completions).
- **Responsive Layout:** Designed to scale beautifully across desktop screens, tablets, and mobile devices.

---

## 🛠️ Built With

* [React.js](https://react.dev/) - Functional Components & React Hooks (`useState`)
* [Vite](https://vitejs.dev/) - Next-generation frontend tooling for blistering fast development
* [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) - Modern Flexbox/Grid structures and bespoke light-mode styling
* Native HTML5 Drag and Drop API
